Machine Builder support given authentication for GitHub:
Credentials (least secure)
Personal access token (medium security)
OAuth (high security, multi-vendor)
GitHub (high security, proprietary)

Security Tips
When using credentials:
Set up a dedicated account(most websites) or dedicated credentials(MediaWiki bot password) and manage it using MachineGuard

When using Personal Access Token:
Set up each PAT for each mod separately. Name them clearly.
